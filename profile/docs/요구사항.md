### 제어
 1. 팬
    - sensorId-Fan01 : 에어컨(온도,습도) ON, OFF
    - sensorId-Fan02 : 환풍기(CO2) ON, OFF
      
 2. 램프
    - sensorId-Lamp01 : 에어컨 온도 내리기
    - sensorId-Lamp02 : 에어컨 온도 올리기
    - sensorId-Lamp03 : 에어컨 제습 모드
    - sensorId-Lamp04 : 에어컨 가습 모드

  3. 부저(있다면?)
     <br>
     - sensorId-alarm01 : 마그네틱
