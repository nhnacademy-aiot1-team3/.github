### 제어
 1. 팬

    - sensorId-Fan02 : 환풍기(CO2) ON, OFF
      
 2. 램프
    - sensorId-Lamp01 : 온도 내리기
    - sensorId-Lamp02 : 온도 올리기

    - sensorId-Lamp04 : 가습모드
