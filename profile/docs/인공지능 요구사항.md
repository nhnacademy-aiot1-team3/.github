## 인공지능 요구사항

1. 목적
    - 온도 제어와 출입 정보 제공을 위한 인공지능
<br>

2. 기능
    - 온도는 그래프를 분석해 희망온도를 예측
    - 출입 카운트는 그래프를 분석해 특정 시간대의 출입 정도를 예측
<br>

3. 성능
    - 정확도는 90% 이상
    - 분류 성능 평가 지표로 나타낼 것
<br>

4. 시스템 요구사항
    - 언어는 python을 사용 할 것
    - 인공지능 모델은 좀 더 생각해볼것 (선형회귀?, ARIMA?)
    - 학습 할 데이터는 influxdb에서 CSV파일형식으로 저장
    - influxdb에서 파일을 가져온다고 하면 데이터 가공 후에 인공지능에 학습
    - 지도학습? 비지도학습??
